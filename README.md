# Splinterlands: Rift Siege

A fan-made tower-defence prototype inspired by Splinterlands.

Deploy familiar cards as towers, exploit Element strengths and weaknesses,
survive multi-lane Rift assaults, discover enemy dossiers, and face legendary
bosses—including Chaos Dragon.

## Play locally

Open [`outputs/index.html`](outputs/index.html) in a modern browser.

## GitHub Pages

This repository includes a Pages workflow. In GitHub, choose:

**Settings → Pages → Source → GitHub Actions**

The game will then receive a public playable URL.

## Highlights

- Five playable Elements with distinct tower mechanics
- Persistent enemy Bestiary with unlockable dossiers
- Splitting, spawning, armored, swift, flying, and fireproof enemies
- Multi-lane maps and five legendary bosses
- Tower evolution, targeting priorities, flawless-wave rewards, and sound
- Automated balance simulation for varied and single-tower strategies
- Responsive fullscreen and landscape-mobile layouts
